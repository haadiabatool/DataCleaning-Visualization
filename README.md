# 🔰DataCleaning-Visualization
Necessary cleaning steps were performed to prepare the data for analysis. After cleaning, exploratory data analysis (EDA) was conducted using different visualizations.

🔰 <b>Dataset Information</b><br>

The dataset includes 6607 records and 20 features, divided into:<br>
<b>➡️Numerical Features</b><br>
•Hours_Studied<br>
•Attendance<br>
•Sleep_Hours<br>
•Previous_Scores<br>
•Tutoring_Sessions<br>
•Physical_Activity<br>
•Exam_Score<br>
<b>➡️Categorical Features</b><br>
•Parental_Involvement<br>
•Access_to_Resources<br>
•Extracurricular_Activities<br>
•Motivation_Level<br>
•Internet_Access<br>
•Family_Income<br>
•Teacher_Quality<br>
•School_Type<br>
•Peer_Influence<br>
•Learning_Disabilities<br>
•Parental_Education_Level<br>
•Distance_from_Home<br>
•Gender<br>

🔰 **Data Cleaning**

The dataset was cleaned to ensure accuracy and consistency:<br>
•Checked for missing values<br>
•Found missing data in:<br>
    🔷Teacher_Quality<br>
    🔷Parental_Education_Level<br>
    🔷Distance_from_Home<br>
•Filled missing values using mode (most frequent value)<br>
•Verified no duplicate records<br>
•Ensured all columns had correct data types<br>

🔰**Data Visualization** <br>

Data visualization was performed to understand the distribution and relationships within the dataset. Histograms were used to analyze numerical features, while scatter plots showed the relationship between different variables and exam scores. Boxplots and bar charts were used to examine the impact of categorical features on student performance.
